Chat Export Formatting Guide

Learn how to export and format your chat data for fine-tuning

WhatsApp Chat Export Guide

  1. Open WhatsApp on your phone
  2. Tap the three dots in the top-right corner
  3. Select Settings
  4. Tap on Chats
  5. Select Chat History
  6. Tap Chat export
  7. Select the chat you want to export
  8. Choose Without Media option
  9. Share the exported file to your computer

The exported file will be in .txt format and will work with our system without any additional formatting.

Discord Chat Export Guide

  1. Use DiscordChatExporter tool
  2. Follow the installation instructions from the GitHub repository
  3. Obtain your Discord authentication token (instructions available in the GitHub repository)
  4. Export your conversations as .txt format
  5. Upload the exported .txt file to our system

Make sure to follow the detailed guide on the DiscordChatExporter GitHub page for specific instructions on token retrieval and export options.

Tips for Better Fine-Tuning Results

  • Export longer conversations for more context and better results
  • Include diverse topics in your training data
  • Make sure the exported text accurately represents your communication style
  • Review the exported data to ensure it doesn't contain sensitive information you don't want in your model
